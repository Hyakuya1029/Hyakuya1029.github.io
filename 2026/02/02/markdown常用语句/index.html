markdown常用语句

发表于2026-02-02|更新于2026-02-04|笔记
|浏览量:

本篇为多方内容和个人使用体会综合而成，仅整理了较为常用的Markdown语法。其中或有缺漏、错误，欢迎联系作者进行查缺补漏。侵权联系必删。
另外，十分推荐Markdown的官方教程，格式与内容更为标准详细。

  • 常用字体效果

    • 基础标题

    效果 语法
    一级标题
    		 # 文字
    二级标题
    		 ## 文字
    三级标题
    		 ### 文字
    四级标题
    		 #### 文字
    五级标题
    		 ##### 文字
    六级标题
    		 ###### 文字

    • 文字修饰

    效果 语法 示例 显示结果
    粗体 **文字** 或 __文字__ **重点内容** 重点内容
    斜体 *文字* 或 _文字_ *补充说明* 补充说明
    粗斜体 ***文字*** ***核心重点*** 核心重点
    删除线 ~~文字~~ ~~作废内容~~ 作废内容
    下划线 <u>文字</u>（MD 原生无，用 HTML 兼容） <u>强调标注</u> 强调标注
    $数学公式$ $文字$（需编辑器支持，如 Typora 等。） \$E=mc²\$ $E=mc²$

    PS：上述行内数学公式的语法使用受到数学公式渲染引擎mathjax的支持
